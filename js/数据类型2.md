# 数据类型
* 原始值类型
* 对象类型
    + 标准普通对象：Object
    + 标准特殊对象：Array、RegExp、



* 普通函数执行过程：
1. 形成一个私有全新的 私有上下文 EC（xx），私有变量对象 AO（xx）
2. 初始化作用域链<EC（xx），作用域>
    初始this
    初始 arg（arguments）
    形参赋值
    变量提升
3. 代码执行
4. 出栈释放和不释放（闭包）

* 箭头函数执行过程：
1. 形成一个私有全新的 私有上下文 EC（xx），私有变量对象 AO（xx）
2. 初始化作用域链<EC（xx），作用域>
    形参赋值
    变量提升
3. 代码执行
4. 出栈释放和不释放（闭包）

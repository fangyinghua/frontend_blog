
### ES2017 (7个特性)

1. Object.values()
2. Object.entries()
3. 字符串追加
String.prototype.padStart和String.prototype.padEnd

4. Object.getOwnPropertyDescriptors
该函数返回指定对象（参数）的所有自身属性描述符
5. 函数参数列表和调用中的尾逗号
```js
 // 函数声明时
    function es8(var1, var2, var3,) {
      // ...
    }
    // 函数调用时
    es8(10, 20, 30,);

```
